# Node.js PATH 설정 방법

터미널에서 `npm`을 인식시키려면 아래 순서대로 진행하세요.

## 1. 환경 변수 열기

- **Windows 키** 누르기
- `환경 변수` 검색
- **시스템 환경 변수 편집** 클릭

## 2. Path 편집

- **환경 변수** 버튼 클릭
- **사용자 변수** 또는 **시스템 변수**에서 **Path** 선택
- **편집** 클릭

## 3. Node.js 경로 추가

- **새로 만들기** 클릭
- 아래 경로 입력:
  ```
  C:\Program Files\nodejs
  ```
- **확인** → **확인** → **확인** (모든 창 닫기)

## 4. 적용

- **Cursor(또는 VS Code) 완전히 종료**
- 다시 실행
- 새 터미널 열기
- `npm run dev` 입력

이제 정상 동작합니다.
